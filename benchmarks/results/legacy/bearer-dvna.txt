Bearer open-source SAST (https://github.com/Bearer/bearer) was not executed in the original CI/workspace snapshot: Docker daemon was not running and official release assets for v2.x do not include a Windows binary.

## Automated reproduction (preferred)

From the repo root, with Docker running and DVNA cloned per `benchmarks/dvna/README.md` (or `./dvna`):

- **PowerShell:** `.\benchmarks\scripts\run-bearer-dvna.ps1`
- **bash:** `./benchmarks/scripts/run-bearer-dvna.sh`

Successful runs write `benchmarks/results/<timestamp>_dvna_bearer/bearer.json`. Summarize theme-aligned true positives into `benchmarks/results/legacy/dvna-evaluation.md` and keep the raw JSON for the paper appendix.

## Manual Docker one-liner

```bash
docker run --rm -v "$PWD/dvna:/scan" bearer/bearer:latest-amd64 scan /scan --format json
```

(Adjust the host path if DVNA lives at `benchmarks/dvna/dvna`.)

## Native install

See https://docs.bearer.com/reference/installation (Linux/macOS packages or install script), then:

```bash
bearer scan ./dvna --format json
```

Record command output under `benchmarks/results/<run>/` for the benchmark table.
