VibeScan

Security scanner npm package for LLM-generated Node.js apps—downstream static analysis, not prompt tricks.

Four contributions

  • Spec-free AST endpoint / route graph for Express without OpenAPI.
  • Persistent .test.js security probes for CI.
  • LLM default-secret dictionary for literals and env fallbacks.
  • Slopsquatting detector (registry check; e.g. RULE-SLOP-001).

Key statistics

10.5% SecPass under security prompting (SusVibes, arXiv:2512.03262)
19.7% hallucinated npm suggestions (Spracklen et al., USENIX Sec 2025)
100% SSRF / 0% CSRF in evaluated AI stacks (Tenzai, Jan 2026)

Real-world impact (one line each)

  • Enrichlead: Cursor-built app, subscription bypass, shut down Mar 2025.
  • Tea App: 72k government ID photos exposed—default Firebase config.
  • CVE-2025-48757: 170 Lovable sites—full DB via missing Supabase RLS.
