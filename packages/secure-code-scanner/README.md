# secure-code-scanner (VibeScan)

Static security scanner for **JavaScript/TypeScript**. Published as npm package `secure-code-scanner`; CLI binaries: **`vibescan`**, **`secure`**.

This package lives in the monorepo under `packages/secure-code-scanner/`. See the repository root [README.md](../../README.md) for full documentation and quick start.

```bash
cd packages/secure-code-scanner
npm install
npm run build
npx vibescan scan ./src
```
